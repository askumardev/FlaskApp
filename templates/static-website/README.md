# Static Website Project

This project is a simple static website that includes multiple HTML pages, a header and footer partial, and linked CSS and JavaScript files.

## Project Structure

```
static-website
├── index.html        # Main entry point of the website
├── home.html         # Another page of the website
├── partials
│   ├── header.html   # Header partial containing site navigation
│   └── footer.html   # Footer partial containing copyright and links
├── css
│   └── styles.css    # CSS styles for the website
├── js
│   └── script.js     # JavaScript functionality for the website
└── README.md         # Documentation for the project
```

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` or `home.html` file in a web browser to view the website.
3. Modify the `css/styles.css` file to change the styles of the website.
4. Update the `js/script.js` file to add or modify JavaScript functionality.

## Usage Guidelines

- The `index.html` file serves as the main entry point and includes the header and footer.
- The `home.html` file is structured similarly to `index.html` and can be used for additional content.
- The `partials/header.html` and `partials/footer.html` files can be updated to change the header and footer across all pages.
- Ensure that all links to CSS and JavaScript files are correctly referenced in the HTML files.
